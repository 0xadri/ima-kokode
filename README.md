# Imakokode

Imakokode stands for "Here and Now" in Japanese. Just had to come up with a name

This is a 4-in-1 Web App, it contains 4 React Mini Apps: 
 1. Project Mgmt App
 2. Tic-Tac-Toe Game
 3. Investment Calculator
 4. Timer Game
